# OpenAPI contract

The running API serves its spec at `/v3/api-docs` (JSON) and `/swagger-ui.html`
(interactive docs), generated by springdoc.

This directory holds the exported spec that the web repo generates its typed
client from (ARCHITECTURE.md §4, "Contract flow"). Export step is a pending
Phase 0 task in [FEATURES.md](../FEATURES.md) — until then, generate against a
running instance:

```sh
curl http://localhost:8080/v3/api-docs > openapi/kindred-api.json
```
